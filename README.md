## Contribution

### Case convention

|                        | convention       | where                                 |
| ---------------------- | ---------------- | ------------------------------------- |
| type, class, interface | PascalCase       | _self-explainatory_                   |
| let                    | lower_snake_case | values that changes                   |
| const                  | UPPER_SNAKE_CASE | values that dont change and shouldn't |
| function               | camelCase        | functions                             |
